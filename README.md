# 🚀 SplitMint — Expense Sharing App

A modern expense-sharing web application inspired by Splitwise, built using Next.js, Prisma, and PostgreSQL. It allows users to manage group expenses, track balances, and simplify bill splitting.

🔗 **Live Demo:** https://splitmint-app-1.onrender.com
💻 **GitHub Repo:** https://github.com/vrushh7/SplitMint-app

---

## ✨ Features

* 👥 Create and share groups
* 💸 Add and track expenses
* ⚖️ Automatic balance calculation
* 🔁 Record reimbursements
* 🧮 Flexible expense splitting
* 📱 Responsive UI with modern design
* 🌙 Dark mode support

---

## 🏗️ Tech Stack

| Layer      | Technology                  |
| ---------- | --------------------------- |
| Frontend   | Next.js, React, TailwindCSS |
| Backend    | Next.js API Routes          |
| Database   | PostgreSQL (Neon)           |
| ORM        | Prisma                      |
| Deployment | Render                      |

## 📦 Installation

```bash
git clone https://github.com/vrushh7/SplitMint-app.git
cd SplitMint-app
npm install
cp .env.example .env
npm run dev
```

---

## 🌐 Deployment

The project is deployed on **Render** with a cloud PostgreSQL database from Neon.
