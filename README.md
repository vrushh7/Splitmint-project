# 🚀 SplitMint — Expense Sharing App

SplitMint is a full-stack expense-sharing application inspired by Splitwise, designed to simplify group expense management and real-time balance tracking.

✨ **Key Features:**

* Group expense management
* Real-time balance calculation (who owes whom)
* Multiple split options (equal/custom)
* Clean and intuitive user interface

## ✨ Features

* 👥 Create and share groups
* 💸 Add and track expenses
* ⚖️ Automatic balance calculation
* 🔁 Record reimbursements
* 🧮 Flexible expense splitting
* 📱 Responsive UI with modern design
* 🌙 Dark mode support

## 🏗️ Tech Stack

| Layer      | Technology                  |
| ---------- | --------------------------- |
| Frontend   | Next.js, React, TailwindCSS |
| Backend    | Next.js API Routes          |
| Database   | PostgreSQL (Neon)           |

## 📦 Installation

```bash
git clone https://github.com/vrushh7/SplitMint-app.git
cd SplitMint-app
npm install
cp .env.example .env
npm run dev
```

---

## 🌐 Deployment

The project is deployed on **Render** with a cloud PostgreSQL database from Neon.
